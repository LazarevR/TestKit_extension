# DevKit — Установка

## Шаг 1 — Сгенерировать иконки

Открой файл `icons/generate.html` в браузере и нажми кнопку.
Три файла (`icon16.png`, `icon48.png`, `icon128.png`) скачаются автоматически —
перемести их в папку `icons/`.

---

## Шаг 2 — Установить расширение

### Chrome / Yandex Browser
1. Открой `chrome://extensions` (или `browser://extensions` в Яндексе)
2. Включи **«Режим разработчика»** (переключатель вверху справа)
3. Нажми **«Загрузить распакованное»**
4. Выбери папку `Extension` (ту, где лежит `manifest.json`)

### Firefox
1. Открой `about:debugging#/runtime/this-firefox`
2. Нажми **«Загрузить временное дополнение»**
3. Выбери файл `manifest.json` внутри папки `Extension`

> Для постоянной установки в Firefox нужна подпись.
> Инструкция: https://extensionworkshop.com/documentation/publish/self-distribution/

---

## Передать коллегам

Просто отправь им папку `Extension` целиком (zip-архив).
Они выполняют те же шаги — никакой сборки не нужно.

---

## Структура

```
Extension/
├── manifest.json          — конфиг расширения
├── background.js          — контекстное меню и логика
├── content.js             — вставка текста на страницу
├── popup/
│   ├── tech-popup.html    — интерфейс анализа технологий
│   └── tech-popup.js      — логика попапа
└── icons/
    ├── generate.html      — генератор иконок (открыть в браузере)
    ├── icon16.png         — иконка 16×16  (после генерации)
    ├── icon48.png         — иконка 48×48  (после генерации)
    └── icon128.png        — иконка 128×128 (после генерации)
```
